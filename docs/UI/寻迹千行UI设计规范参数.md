# 寻迹千行 - UI 设计规范参数表

> 供 UI 设计师参考，所有页面视觉设计需遵循以下规范。
> 设计尺寸基准：**750rpx（375pt @2x）**

---

## 一、品牌色彩体系

### 1.1 主色调

| 色彩角色 | 色值（建议） | 用途说明 |
|---------|-------------|---------|
| 品牌主色 | `#FF6B35`（暖橙色） | 按钮、价格、Tab 选中态、重点文字强调 |
| 品牌主色-浅 | `#FFF3ED` | 标签背景、轻提示背景 |
| 品牌主色-深 | `#E55A2B` | 按钮按下态 |

> **说明**：文档要求「旅游/户外感，建议暖色系」。推荐暖橙色，设计师可在 `#FF6B35` ~ `#FF8C00` 区间微调，也可考虑珊瑚橙 `#FF7F50` 或落日橙 `#F4845F`。最终由设计师出 2-3 套色板供选择。

### 1.2 中性色 / 文字色

| 色彩角色 | 色值（建议） | 用途说明 |
|---------|-------------|---------|
| 标题文字 | `#1A1A1A` | 页面标题、卡片标题、重要信息 |
| 正文文字 | `#333333` | 正文内容、列表主文字 |
| 次要文字 | `#666666` | 辅助说明文字、时间、标签说明 |
| 占位/提示文字 | `#999999` | 输入框 placeholder、禁用态文字 |
| 禁用文字 | `#CCCCCC` | 不可点击、过期态 |

### 1.3 状态色

| 状态 | 色值（建议） | 应用场景 |
|------|-------------|---------|
| 红色-危险/紧急 | `#FF4D4F` | 待支付、待处理、价格标红、删除 |
| 黄色-警告/等待 | `#FAAD14` | 待确认、跟进中、审核中 |
| 蓝色-信息/进行 | `#1890FF` | 待出行、进行中、链接 |
| 绿色-成功/完成 | `#52C41A` | 已完成、已通过、可用、支付成功 |
| 橙色-处理中 | `#FF8C00` | 退款中、处理中 |
| 灰色-失效/关闭 | `#BFBFBF` | 已取消、已关闭、已过期、已退款 |

### 1.4 背景色

| 色彩角色 | 色值 | 用途说明 |
|---------|------|---------|
| 页面背景 | `#F5F5F5`（文档明确） | 所有页面默认背景 |
| 卡片背景 | `#FFFFFF` | 卡片、列表项、弹窗背景 |
| 分割线 | `#EEEEEE` | 列表分隔线、区块分隔 |
| 遮罩层 | `rgba(0,0,0,0.5)` | 弹窗、底部弹层遮罩 |

## 二、字体规范

### 2.1 字体族

| 平台 | 字体 |
|------|------|
| iOS | PingFang SC |
| Android | 系统默认（思源黑体 / Roboto） |
| 数字/价格 | DIN Alternate / DIN Pro（推荐） |

### 2.2 字号体系

| 层级 | 字号（rpx） | 字号（pt） | 字重 | 应用场景 |
|------|-----------|-----------|------|---------|
| H1 大标题 | 40rpx | 20pt | Bold (700) | 页面主标题（极少用） |
| H2 标题 | 36rpx | 18pt | Semibold (600) | 导航栏标题、区块标题（如"热门线路"） |
| H3 副标题 | 32rpx | 16pt | Medium (500) | 卡片标题、线路名称、表单标签 |
| 正文-大 | 30rpx | 15pt | Regular (400) | 正文主内容、按钮文字 |
| 正文 | 28rpx | 14pt | Regular (400) | 列表主文字、输入框文字 |
| 辅助文字 | 24rpx | 12pt | Regular (400) | 时间、标签、说明文字、Tab 文字 |
| 小字 | 22rpx | 11pt | Regular (400) | 版权信息、极次要说明 |
| 价格-大 | 48rpx | 24pt | Bold (700) | 线路详情页价格 |
| 价格-中 | 36rpx | 18pt | Bold (700) | 卡片价格、订单金额 |
| 价格-小 | 28rpx | 14pt | Semibold (600) | 日历价格、明细金额 |

### 2.3 行高

| 场景 | 行高比例 |
|------|---------|
| 标题 | 1.3 倍 |
| 正文 | 1.5 倍 |
| 多行说明 | 1.6 倍 |

| 组件 | 圆角值（rpx） | 说明 |
|------|-------------|------|

## 四、间距与布局

### 4.1 页面级间距

| 间距类型 | 值（rpx） | 说明 |
|---------|----------|------|
| 页面左右边距 | 24~32rpx | 页面内容到屏幕两侧的距离 |
| 区块间距 | 20~24rpx | 不同功能区块之间的纵向间距 |
| 卡片内边距 | 24rpx | 卡片内容到卡片边缘的距离 |
| 列表项间距 | 16~20rpx | 列表中各卡片之间的纵向间距 |

### 4.2 宫格布局（首页功能入口）

| 参数 | 值 |
|------|------|
| 行列 | 2 行 × 4 列 |
| 图标尺寸 | 80~96rpx |
| 图标与文字间距 | 8~12rpx |
| 单元格间距 | 均分等宽 |

### 4.3 瀑布流布局（首页热门线路）

| 参数 | 值 |
|------|------|
| 列数 | 2 列 |
| 列间距 | 16~20rpx |
| 行间距 | 16~20rpx |

### 4.4 底部安全区

| 参数 | 说明 |
|------|------|
| TabBar 高度 | 遵循微信小程序默认 |
| 底部固定操作栏 | 高度约 100~120rpx + 底部安全区 |

## 五、按钮规范

### 5.1 主按钮（CTA）

| 属性 | 值 |
|------|------|
| 背景色 | 品牌主色 `#FF6B35` |
| 文字色 | `#FFFFFF` |
| 字号 | 30~32rpx / Medium |
| 高度 | 88rpx |
| 宽度 | 通常为满宽（减去左右边距） |
| 圆角 | 44rpx（全圆角） |
| 按下态 | 背景色加深 `#E55A2B` |
| 禁用态 | 背景色 `#FFD1B8`，文字 `#FFFFFF` |

> 应用场景：立即预订、提交订单、微信一键登录、保存、提交申请等

### 5.2 次要按钮

| 属性 | 值 |
|------|------|
| 背景色 | `#FFFFFF` |
| 边框 | 1px solid 品牌主色 |
| 文字色 | 品牌主色 |
| 高度 | 72rpx |
| 圆角 | 36rpx |

> 应用场景：返回首页、取消订单、联系客服等

### 5.3 文字按钮 / 小操作按钮

| 属性 | 值 |
|------|------|
| 文字色 | 品牌主色 或 `#1890FF` |
| 字号 | 24~28rpx |
| 无背景 | 仅文字，点击态加下划线或变淡 |

> 应用场景：查看详情、编辑、删除、去使用等

### 5.4 危险按钮

| 属性 | 值 |
|------|------|
| 背景色 | `#FFFFFF` |
| 边框 | 1px solid `#FF4D4F` |
| 文字色 | `#FF4D4F` |

> 应用场景：取消需求、申请退款等

## 十四、设计风格关键词

> 供设计师把握整体设计方向

- **温暖亲和**：暖色系主色调，传递旅游的愉悦感
- **简洁清爽**：浅色背景 + 白色卡片，信息层次分明
- **信任专业**：规范的信息展示，清晰的状态反馈
- **微信生态统一**：遵循微信小程序设计规范，原生组件优先
- **图片驱动**：旅游线路以大图吸引，注重图片质量和展示比例

## 十五、设计交付清单

设计师需输出以下内容：

1. **色板确认稿** — 主色 2-3 套方案供选择
2. **组件库/设计系统** — 按钮、卡片、表单、标签、弹窗等基础组件
3. **图标库** — TabBar 图标（选中/未选中）、功能入口图标、状态图标
4. **全部页面设计稿** — 共 35 个页面（含各状态）
5. **标注文件** — 间距、字号、色值标注
6. **切图资源** — @2x / @3x
